Netflix Data Analysis Dashboard (Tableau)
Project Overview

The Netflix Data Analysis Dashboard is an interactive visualization project created using Tableau to analyze Netflix’s movies and TV shows catalog. The dashboard provides insights into Netflix content distribution by country, genre, ratings, and release trends over time.

This project aims to transform raw Netflix data into meaningful visual insights that help understand the growth of Netflix content, popular genres, and global distribution of movies and TV shows.

The dashboard demonstrates data visualization, exploratory data analysis, and storytelling using Tableau, which are essential skills for a Data Analyst role.

Key Insights
Content Distribution

Movies make up 68.42% of Netflix content.

TV Shows account for 31.58% of the catalog.

Ratings Analysis

The most common rating category is TV-MA, followed by TV-14.

Mature audience content dominates the Netflix library.

Country Contribution

The United States has the highest number of Netflix titles.

Other major contributors include India, the United Kingdom, and Canada.

Top Genres

The most popular genres on Netflix include:

Documentaries

Stand-Up Comedy

Dramas & International Movies

Kids’ TV

Children & Family Movies

Content Growth

Netflix content significantly increased between 2016 and 2019.

Both movies and TV shows saw rapid expansion during this period.
